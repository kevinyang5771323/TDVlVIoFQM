# Python-25010 项目简介 python789

本项目是基于 Flask、Vue.js 和 MySQL 技术栈开发的人力资源信息系统。系统以简洁的界面和实用的功能，为用户提供了高效便捷的点对点管理模式。

## 技术栈

- 后端：Python（Flask 框架）
- 前端：Vue.js
- 数据库：MySQL

## 功能模块

- **权限认证**：使用 JWT（JSON Web Token）进行安全认证。
- **角色管理**：对系统角色进行管理，分配权限。
- **用户管理**：管理用户信息，分配角色。

## 系统角色

- 系统管理员：负责系统的整体管理和维护。
- 普通用户：根据角色权限进行相应操作。

## 运行环境

- Python 3.*
- Flask 1.*
- Vue.js 2.*
- MySQL 5.*

## 部署步骤

1. 环境准备：确保 Python、MySQL 等基础软件安装完备。
2. 数据库初始化：根据项目提供的 SQL 文件进行数据库初始化。
3. 配置后端：配置 Flask 应用所需的数据库连接及 JWT 参数。
4. 部署前端：将 Vue.js 前端项目编译后的文件放置到指定目录。
5. 运行后端：启动 Flask 应用。
6. 访问前端：通过浏览器访问部署好的前端页面。

## 目录结构

```
project/
```

## 核心亮点

- **安全性**：使用 JWT 进行权限认证，确保系统的安全性。
- **模块化设计**：清晰的模块划分，便于维护和扩展。
